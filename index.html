Garage 94

Descripción general

Aplicación web de e-commerce musical desarrollada con Vue, enfocada en la exploración de productos como CDs, vinilos y mercadería, con una interfaz dinámica inspirada en la cultura del rock alternativo.

¿Qué hace?

Permite al usuario navegar por un catálogo de productos musicales mediante un sistema de paginación dinámica, ofreciendo una experiencia fluida e interactiva en la exploración del contenido.

Tecnologías utilizadas
  • Vue.js
  • JavaScript
  • HTML5
  • CSS3
  • Vercel (deploy)
Lo que desarrollé
  • Renderizado dinámico de productos con Vue
  • Sistema de paginación controlado por estado
  • Manejo de eventos y reactividad del framework
  • Organización de componentes
  • Diseño alineado a temática musical
Retos y soluciones
  • Gestión del estado de paginación: Uso de la reactividad de Vue para actualizar contenido dinámicamente
  • Navegación sin recarga: Implementación de lógica SPA
  • Escalabilidad: Separación en componentes reutilizables
Aprendizajes clave
  • Uso práctico de Vue en aplicaciones dinámicas
  • Manejo de estado en frontend
  • Creación de interfaces reactivas
  • Organización basada en componentes

Funko Memory Game

Descripción general

Aplicación web interactiva desarrollada con React que simula un juego de memoria basado en cartas tipo Funko Pop. Incluye autenticación de usuarios, efectos de sonido y personalización de la experiencia mediante selección de dificultad y categorías.

¿Qué hace?

El usuario inicia sesión y juega a encontrar pares de cartas en el menor número de movimientos posible. El juego gestiona el estado en tiempo real, reproduce efectos de sonido y permite adaptar la dificultad, ofreciendo una experiencia dinámica e inmersiva.

Tecnologías utilizadas
  • React
  • JavaScript (ES6+)
  • HTML5
  • CSS3
  • Firebase (autenticación y base de datos)
Lo que desarrollé
  • Implementación de lógica completa del juego de memoria
  • Sistema de login y gestión de usuarios
  • Manejo de estado en React (cartas, turnos, coincidencias)
  • Integración de efectos de sonido para mejorar la experiencia
  • Selección dinámica de dificultad y categorías
  • Persistencia de datos con Firebase
  • Renderizado condicional de componentes
Retos y soluciones
  • Gestión del estado del juego en tiempo real: Uso de React para controlar cartas, turnos y validaciones
  • Integración de autenticación de usuarios: Implementación de Firebase Auth para manejo de sesiones
  • Sincronización de sonidos con acciones del usuario: Control de eventos para reproducir audio sin afectar rendimiento
Aprendizajes clave
  • Manejo avanzado de estado en React
  • Integración de autenticación en aplicaciones frontend
  • Mejora de UX mediante feedback auditivo
  • Consumo y gestión de servicios backend (Firebase)

Pensá en Verde
Resumen del Proyecto

Pensá en Verde es un afiche ficticio de campaña ecológica diseñado para incentivar el reciclaje mediante una recompensa atractiva. Se emplea una estética fresca, amigable y clara para captar al público urbano joven. El diseño se enfoca en motivar la acción inmediata, conectando el reciclaje con un evento cultural sostenible.

  • Verde hoja (#4CAF50) – Principal
  • Verde lima claro (#d2e27f) – Fondo
  • Verde oscuro (#1E5E2F) – Tipografía secundaria
  • Amarillo vibrante (#FFEB3B) – Detalles o énfasis
  • Tipo: Diseño de afiche promocional
  • Herramientas: Photoshop, Illustrator
  • Tema: Conciencia ambiental
  • Fuente: Monserrat
Asfalto
Resumen del Proyecto

Asfalto es una revista ficticia dedicada a la cultura del skateboarding urbano. La portada fue diseñada para capturar la energía de las calles, utilizando una composición dinámica. El enfoque visual busca reflejar la actitud desafiante del skate, conectando con una audiencia joven, alternativa y apasionada por el movimiento, la libertad y la autogestión.

Paleta de colores sugerida:

  • Azul marino oscuro (#01306A) – Fondo
  • Blanco (#FFFFFF) – Para el logo y texto
  • Amarillo brillante (#FFFE00) – texto
  • Rojo Carmesí (#BA0B1A) – fondo de texto
  • Azul Oscuro (#100E6F) – fondo de texto
  • Tipo: Diseño de portada editorial
  • Herramientas: Photoshop
  • Tema: Cultura urbana y skateboarding
